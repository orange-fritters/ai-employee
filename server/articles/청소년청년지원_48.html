대학생 연합생활관(은행권, 고양)

1. 대상 국내 소재 대학(원) 소속 학생(휴학생 및 외국인도 입주 가능)

2. 내용 ∙ 위치 : 경기도 고양시 덕양구 원흥1로 23(지하철 3호선 원흥역 인근) ∙ 수용인원 : 총 986명(493실, 장애인 5실 별도) ∙ 기숙사비 : 월 15만원(2인 1실) / 보증금 15만 원(퇴실 시 전액 환급) ∙ 주요 편의시설 : 식당, 카페, 편의점, 세탁실, 도서관, 멀티미디어실, 열람실, 체력단련실, 공동취사실

3. 방법 ∙ 모집시기 - 정기모집 : 1월(1학기), 7월(2학기 예정) - 수시모집 : 공실 발생 시 매월 첫째 주 수시모집 예정 ∙ 신청방법 : 한국장학재단 누리집에서 온라인 신청

4. 문의 한국장학재단(☎1599-2290, www.kosaf.go.kr)
